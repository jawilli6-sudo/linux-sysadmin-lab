# Linux SysAdmin Lab

Hands-on RHEL administration projects documenting my transition into 
Linux systems administration and federal IT security.

Every project in this lab incorporates DISA STIG compliance — the 
security hardening standard used by the U.S. Department of Defense to 
protect government systems. Security is built in from the start, not 
added as an afterthought.
